Install youtube-material with npm

```bash
  npm install youtube-material
  cd youtube-material
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/livingdolls/youtube-material.git
```

Go to the project directory

```bash
  cd youtube-material
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
